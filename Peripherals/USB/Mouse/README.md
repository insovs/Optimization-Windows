# Mouse

Collection of registry tweaks and scripts to optimize mouse input on Windows — reduces latency, improves responsiveness and removes unwanted system behaviors.

## Files
| File | Description |
|---|---|
| `Main-Mouse-Optimizations.bat` | Applies all main mouse tweaks at once |
| `DisableCursorMagnetism.reg` | Disables cursor magnetism on dialog boxes |
| `MouseTransmitTimeout.reg` | Tweaks mouse transmit timeout for lower latency |
| `PollStatusIterations.reg` | Adjusts poll status iterations |
| `CursorUpdateInterval/` | Controls cursor update rate — see subfolder README |

## Usage
1. Run `Main-Mouse-Optimizations.bat` as administrator for a quick setup.
2. Apply individual `.reg` files manually depending on what you need.
