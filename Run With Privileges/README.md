# PowerRun – Run as TrustedInstaller

**PowerRun** lets you run programs or scripts with **TrustedInstaller** privileges — the highest level in Windows.
This grants access to protected system files and registry keys that even Administrators cannot normally modify.

## Usage

1. Open **PowerRun.exe** and enable **“Allow Command Lines”** in the *File* menu if not already enabled.
2. Drag and drop the script you want to run into PowerRun.
   It will execute with maximum privileges, avoiding errors, access denials, and system restrictions.
