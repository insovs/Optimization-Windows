# Privacy

Services that track your location or share your bandwidth without you knowing.

| File | What it disables | Watch out |
|------|-----------------|-----------|
| `Disable-GeolocationService.reg` | Location access for all apps | Automatic timezone detection will stop working |
| `Disable-DeliveryOptimization.reg` | Uses your internet to distribute Windows updates to other PCs | — |
| `Disable-RetailDemo.reg` | Windows store demo mode | — |
